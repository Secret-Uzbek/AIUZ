## What this repository is

`AIUZ` is not just a software stub and not just a raw archive dump.

It currently acts as a mixed repository containing:

- AIUZ concept documents;
- educational and ethics materials;
- dictionary and language-learning components;
- grant, dossier, and program-supporting files;
- repository infrastructure for turning this branch into a more coherent public
  surface.

## Current status

This repository is in normalization.

That means:

- useful source materials are already here;
- some files still come from mixed drafting layers;
- service-facing files are being rewritten to become readable for humans again;
- the repository still needs deeper structural cleanup below the root level.
