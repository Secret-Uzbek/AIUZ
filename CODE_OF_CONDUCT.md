# Code of Conduct

`AIUZ` is a repository for ethical educational technologies and related concept
materials. That means conduct here applies not only to code, but also to
documentation, learning content, language materials, grant-facing texts, and
AI-assisted edits.

## Purpose

Participation in this repository should remain:

- respectful;
- non-harassing;
- readable for humans;
- aligned with child safety and educational integrity;
- honest about what is draft, donor-derived, or machine-assisted.

## Expected behavior

- discuss ideas without personal attacks;
- preserve educational and ethical integrity when editing content;
- be explicit when a file is experimental, operational, or machine-facing;
- avoid adding noise, broken encoding, or misleading placeholders to visible
  repository surfaces.

## Unacceptable behavior

- harassment, intimidation, or degrading language;
- publishing credentials, private data, or sensitive child-related material;
- disguising unfinished AI output as finished human documentation;
- using the repository for exploitative or manipulative educational patterns.
