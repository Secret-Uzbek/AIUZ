# Contributing to AIUZ

`AIUZ` is part of the wider Terra / FMP ecosystem.

This repository should be improved in ways that make it:

- more readable;
- more ethically coherent;
- less fragmented;
- and safer for educational/public interpretation.

## Before contributing

Please read:

- `README.md`
- `CODE_OF_CONDUCT.md`
- `SECURITY.md`
- the donor legal layer in `AIUZ-Terra-Codex-EcoSystem/terra-legal`

## Good contribution types

- README and documentation repair
- mojibake and formatting cleanup
- ethics and governance clarification
- educational structure cleanup
- branch-specific public landing pages
- citation and publication metadata repair

## Avoid

- adding noisy automation residue to public surfaces;
- committing credentials or personal secrets;
- presenting machine-generated placeholders as finished human texts;
- treating archive residue as if it were already canonical.

## Pull request rule

Changes should explain:

- what human-facing problem they fix;
- whether the affected file is public, donor, technical, or machine-facing;
- whether they improve readability, integrity, or public trust.
