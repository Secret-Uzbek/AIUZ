# Security Policy

This file describes the repository-specific security boundary for `AIUZ`.

## What belongs here

Report issues that affect:

- repository integrity;
- public-facing documentation or pages;
- accidental exposure of private or sensitive educational material;
- scripts, workflows, or automation files stored in this repository.

## Reporting

Please report security concerns to:

- `a.abdukarimov@fractal-metascience.org`

Do not open public issues for exploitable vulnerabilities or exposed secrets.
